yt-grab: Simple Video & Audio Downloads

Download with a clean command flow: paste a link, choose audio or video, and set quality quickly.

Open Repository Quick Commands

Video: yt-grab https://youtu.be/8ekJMC8OtGU --quality 1080p

Audio: yt-grab https://youtu.be/8ekJMC8OtGU --audio

Direct Mode

Run downloads directly as yt-grab <url> without learning subcommands.

Audio Output

Use --audio and optional --audio-format for mp3, aac, or wav.

Quality Flags

Set --quality best|worst|720p|1080p for quick control over output quality.

Cross Platform

Works on Linux, macOS, and Windows PowerShell with clear command examples.
